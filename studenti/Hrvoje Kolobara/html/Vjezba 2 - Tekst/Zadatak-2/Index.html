Uvod u HTML

HTML je jezik za obiljezavanje koji kostistimo za izradu web stranica.
Njime definiramo strukturu i sadrzaj web stranica.

Povijest HTML-a

HTML je nastao početkom 1990-ih godina.

Uredjivanje teksta

Tekst je najcesci sadrzaj web stranica.
Da bi dobio odgovarajucu ulogu na stanici i bio prikazan na odredjeni nacin uredjujemo ga HTML-om.

Html Atributi

Unutar početne oznaka elemenata, nakon naziva elementa, po potrebi pišemo i atribute.
Atributi internetskim preglednicima daju dodatne informacije o elementu. O samom atributu ovisi o kojim informacijama je riječ.
Atribut se sastoji od imena i vrijednosti atributa.
