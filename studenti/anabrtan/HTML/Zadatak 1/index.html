Rad s grafikom u HTML-u

Formati grafičkih datoteka

Za prikaz grafike na webu koriste se sljedeći formati:

JPEG
Joint Photograpic Experts Group
Koristi se za prikaz fotografija
GIF
Graphics Interchage Format
Koristi se za prikaz jednostavne grafike (crteži, ilustracije, i sl.)
PNG
Portable Network Graphics
Koristi se za prikaz fotografija i crteža uz mogućnost transparencije
SVG
Scalable Vector Graphics
Koristi se za prikazivanje dvodimenzionalnih vektorskih slika (ikone, logotipi, i sl.)
