Uvod u HTML

HTML je jezik za obilježavanje koji koristimo za izradu web stranica.
Njime definiramo strukturu i sadržaj web stranica.

Povijest HTML-a

HTML je nastao početkom 1990-ih.

Uređivanje teksta

Tekst je najčešći sadržaj web stranica.
Da bi dobio odgovarajuću ulogu na stranici i bio prikazan na određeni način uređujemo ga HTML-om.

HTML atributi

Unutar početne oznake elemenata, nakon naziva elementa, po potrebi pišemo i atribute.
Atributi internetskim preglednicima daju dodatne informacije o elementu. O samom atributu ovisi o kojim je informacijama riječ.
Atrubut se sastoji od imena i vrijednosti atributa.
